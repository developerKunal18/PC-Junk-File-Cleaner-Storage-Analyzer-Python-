# PC-Junk-File-Cleaner-Storage-Analyzer-Python-
A Python program that: ✔ Scans the system for junk / temporary files ✔ Shows total junk size ✔ Deletes selected junk safely ✔ Shows top folders consuming storage ✔ Works on Windows / Mac / Linux ✔ No admin privileges required  This is like a mini CCleaner written in Python.
